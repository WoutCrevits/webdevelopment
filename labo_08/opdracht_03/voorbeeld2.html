Tekst staat horizontaal gecentreerd door een door 'text-align: center' bij je div element in je stylesheet.

'center' is gen geldige waarde voor vertical-align.

'middle' zal ervoor zorgen dat de div-elementen centraal in de regel staan.

'height: 200px; line-height: 200px;' Als height en line-height hetzelfde zijn staat de tekst gecentreerd.

'.container {display:table; height:100px} .text{display:table-cell; vertical-align:middle;}' om meerdere regels te centreren.

'.tall' staat later in de CSS file dan '.area'
